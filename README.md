# Donation Tracking Dashboard

This is a backend-based donation tracking dashboard built using Django.  
It shows total donations received, number of unique donors, donation trends using a chart, and a table of recent donations.

---

## Features

- Displays total amount donated
- Shows number of unique donors
- Donation trend chart using Chart.js
- Responsive UI using Bootstrap
- Shows recent donations in a table
- Uses dummy dataset for demonstration

---

## Tech Stack

- Django (Backend)
- SQLite (Database)
- HTML, CSS, Bootstrap, Chart.js (Frontend)
